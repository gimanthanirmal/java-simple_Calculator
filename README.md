# Java Console Calculator App

A command-line calculator written in Java that supports basic arithmetic operations, input validation, history tracking, and graceful termination.  
This project demonstrates **object-oriented programming**, **user input handling**, and **program control flow** in Java.

---

## ✨ Features
- Addition, subtraction, multiplication, and division
- Input validation with error handling
- Prevents division by zero
- Reset (`!`) and terminate (`#`) commands
- History of the last 5 results (`@`)
- Scheduled exit with countdown

---

## 📂 Project Structure

CalculatorApp.java   # Main entry point
MathProcessor.java   # Arithmetic operations
ScheduledExit.java   # Delayed termination logic
CalStart.java        # Menu, input handling, history
